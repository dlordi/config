# Wave Terminal

- Wave Terminal configuration docs: https://docs.waveterm.dev/config

- create a symlink for the configuration file
  - on Windows, the configuration file is `%USERPROFILE%\.config\waveterm\settings.json`

    ```bat
    mklink "%USERPROFILE%\.config\waveterm\settings.json" "%PATH_TO_THIS_REPO%\waveterm\settings.json"
    ```
